# Portfolio

Modern React portfolio for Lou-Junel T. Velasco.

Public URL target:

```text
https://loujunel.github.io/portfolio/
```

## Stack

- Vite
- React
- lucide-react
- Static GitHub Pages deployment

## Content Rules

The site uses public-safe portfolio notes as the source of truth. Do not publish phone number, full address, references, salary, government IDs, raw scans, private records, or unconfirmed personal details.

## Local Development

Install dependencies:

```powershell
pnpm install
```

Run locally:

```powershell
pnpm dev
```

Build:

```powershell
pnpm build
```

Preview the production build:

```powershell
pnpm preview
```

## GitHub Pages

This is a project site, so Vite is configured with:

```js
base: "/portfolio/"
```

Publish the built site to GitHub Pages from the `loujunel/portfolio` repository.
